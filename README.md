# StoreParticleExtension
Adds Particle Systems to ZephStore

Item.txt Examples
```javascript
"My Spawn Effect"
{		
			"Particle" "myParticleName"	
			"price" "4711"
			"type" "spawn_effect"
			"Duration" "1.5"
}

"My Hit Effect"
{		
			"Particle" "myParticleName"	
			"price" "4711"
			"type" "hit_effect"
			"Duration" "1.5"
}

"My Kill Effect"
{		
			"Particle" "myParticleName"	
			"price" "4711"
			"type" "kill_effect"
			"Duration" "1.5"
}

"My Aura"
{
				"Name" "myParticleName"
				"price" "10000"
				"type" "Aura"
}

"My Trail"
{
				"Name" "myParticleName"
				"price" "10000"
				"type" "Particles"
}
```
